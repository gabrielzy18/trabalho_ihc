Este projeto contempla informações sobre os candidatos a prefeito do município de Santos, localizado no estado de São Paulo, das Eleições que acontecerão no ano de 2020.
Período das eleições 2020: 15/11/2020 (domingo) - 29/11/2020 (domingo)












Este projeto foi realizado para complementar a nota da disciplina Interação Humano-Computador-II da Universidade Católica de Santos, tendo como docente o Prof. Me. Thiago Ferauche.
